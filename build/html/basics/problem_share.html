分享试题¶

进入 试题详情 页面( 见 查看试题详情 )，点击 分享/导出 下拉菜单.

分享到朋友圈¶

选择 分享到朋友圈，手机打开微信扫描二维码后即可分享试题链接到朋友圈.

分享到微信公众号¶

选择 复制到微信公众号，弹出预览窗口.

滚动到弹出窗口的底部，选择需要的 样式 ( 见 自定义css样式 ).

点击弹窗底部 复制到微信公众号 按钮，前往微信公众号图文编辑页面 ctrl+v 粘贴即可.

分享到第三方编辑器¶

为了满足用户更多地定制化需求，橘子数学也支持用户将试题车导出为 markdown 或 \(\TeX\) 分享到第三方编辑器进行编辑.

推荐的 markdown 编辑器：

  • Typora: 最像word的 markdown 编辑器.

  • Markdown Nice: 优秀的国内开源在线 markdown 编辑器.

  • StackEdit: StackExchange 出品的开源在线 markdown 编辑器.

推荐的 \(\TeX\) 的编辑器:

  • Overleaf: 国外流行的在线 \(\TeX\) 编辑器，可以实现在线编译.

  • TeXStudio: 最好用的 \(\TeX\) ，需要安装 TeXLive 才能正常编译.

警告

由橘子数学导出的 \(\TeX\) 代码中包含大量自定义环境，建议使用 BHCexam 宏包进行编译.
